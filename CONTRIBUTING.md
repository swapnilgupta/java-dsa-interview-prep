# Contributing

Thanks for considering contributing!

## How to Contribute

1. **Fork** the repository
2. **Create** a branch: `git checkout -b add-problem-name`
3. **Add** your solution in the appropriate folder
4. **Test** your code
5. **Commit**: `git commit -m "Add [Problem Name] solution"`
6. **Push** and create a **Pull Request**

## Code Standards

### File Naming
Use PascalCase: `LongestIncreasingSubsequence.java`

### Code Template
```java
package <category>;

/**
 * Problem: [Name]
 * LeetCode: [URL]
 * Difficulty: Easy/Medium/Hard
 * 
 * Time: O(?)
 * Space: O(?)
 */
public class ProblemName {
    
    public ReturnType solve(params) {
        // Solution with comments
    }
    
    public static void main(String[] args) {
        // Test cases
    }
}
```

## Folder Structure

Place solutions in the correct category:
- `arrays/` - Array problems
- `dp/` - Dynamic Programming
- `graph/` - Graph algorithms
- `tree/` - Tree problems
- `backtracking/` - Backtracking
- `string/` - String manipulation
- etc.
